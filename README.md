# DLBDSPBDM01 – Buchtausch-App Datenbank

## Projektbeschreibung
Dieses Repository enthält die Ergebnisse des IU Portfolio-Kurses „Data-Mart-Erstellung in SQL“. Ziel des Projekts ist die Konzeption und Implementierung einer relationalen Datenbank für eine Buchtausch-App, die den Austausch und die Ausleihe von Büchern innerhalb einer lokalen Gemeinschaft ermöglicht.

## Technologien
- MySQL 9.x
- InnoDB Storage Engine
- UTF8MB4 Zeichensatz
- MySQL Workbench

## Projektstruktur

01_Konzeptionsphase  
Enthält das Konzeptionsdokument mit Anforderungsspezifikation, Problembeschreibung, ER-Modell und Datenwörterbuch.

02_Erarbeitungsphase  
Enthält die SQL-Implementierung der Datenbank sowie die Präsentation der Implementierungsschritte und Testabfragen.

03_Finalisierung  
Enthält die technische Dokumentation des Systems und das Abstract der Arbeit.

## Installation

1. SQL-Datei aus dem Ordner **02_Erarbeitungsphase** herunterladen.
2. Datei in MySQL Workbench öffnen.
3. Das Skript vollständig ausführen.
4. Die Datenbank `buchtausch_app` wird automatisch erstellt.
5. Dummy-Daten und Testabfragen sind bereits enthalten.
